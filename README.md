# Trabajo Práctico 3 - Portfolio

Lara De Luca
Computación III Diseño y Comunicación Visual 

Desarrollo de un portfolio personal e interactivo que se adapta tanto a computadoras de escritorio como a celulares (hasta 600px). 
En la sección "Sobre Mí" sumé un efecto de hover con sonido al pasar por las cartas para hacerlo más dinámico. 
También armé una grilla para mostrar los trabajos recientes y rediseñé el formulario de contacto para que sea cómodo de completar desde el celular.


Detalles Técnicos
Editor de CÓDIGO: Visual Studio Code
Resolución de pantalla: 1920x1080
